* classnames
* react-router-dom
* ESLint
* Prettier

## How to install

1. Clone the repo:

   ```sh
   git clone https://github.com/dmtrhrytsak/crypto_kitties.git
   ```
2. Install packages:

   ```sh
   npm install
   ```
3. Run the app

   ```
   npm start
   ```
